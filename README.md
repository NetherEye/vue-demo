## 项目运行（nodejs 6.0 以上版本）


## Build Setup

``` bash

# 安装依赖
npm install

# 开启本地服务器localhost:8088
npm run dev

# 发布环境
npm run build

# 其他辅助功能（包括单元测试等）

# build for production and view the bundle analyzer report
npm run build --report

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
